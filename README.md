# DevShop E-commerce Project

## Overview
DevShop is a simple e-commerce website built using **HTML, CSS, and Bootstrap**. Users can browse products, view details, and add them to a cart. This project focuses only on the frontend design.

---

## Features
- Responsive layout for desktop and mobile
- Home page, product listing, and product details
- Shopping cart UI
- Navigation bar and footer
- Clean design using Bootstrap components

---

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
